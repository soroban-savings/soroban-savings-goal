# Soroban Savings Goal Contract

A simple Soroban smart contract that allows users to:

- Create savings goals
- Deposit funds
- Track balance
- Withdraw when target is reached

## Functions

### create_goal(target)

Creates a savings goal.

### deposit(amount)

Adds funds to the goal.

### get_balance()

Returns current balance.

### get_target()

Returns target amount.

### withdraw()

Withdraws funds after reaching target.

## Tech Stack

- Rust
- Soroban SDK
- Stellar
